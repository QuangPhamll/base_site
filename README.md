# base_site
A sample projet of full-stack web app with wagtail postgres for backend and nuxtjs vue for frontend.
Version:
    - django: v.2.2
    - wagtail: v.2.5
    - postgres: 11
    - nuxjs: v.2.8
# Run app:
docker-compose build
docker-compose up
backend cms: http://localhost:7999/admin
fronttend: http://localhost:8000
